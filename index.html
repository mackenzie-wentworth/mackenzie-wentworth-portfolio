Peak Fitness

Visit GitHub Repo & Deployed Project for more details.

A front-end web application for users to search bike trails and the corresponding weather forecast for a particular location.

This was a group project crafted with the combination of HTML, JavaScript, CSS, Bulma CSS framework, and JQuery to render up-to-date data retrieved from two server-side APIs and using client-side storage to store persistant data. Our choice of server-side APIs include the OpenWeather API for displaying current weather data and the TrailAPI to display a list of trails for users to explore. As part of my contributions to my team, I was able to showcase my front-end abilities through API integration and deliver a user-intuitive application that caters to outdoor enthusiasts seeking biking trails and current weather conditions.

Code Thread

Visit GitHub Repo & Deployed Project for more details.

A full-stack e-commerce website for Coding Boot Camp graduates to browse and buy cohort t-shirts. CodeThread was a team collaborative project motivated by the spirit of our boot camp to show off our cohort accomplishments in style.

This web app was built from scratch following the Model-View-Controller (MVC) paradigm and features the following technologies: HTML, JavaScript, CSS, Node.js, Express.js, Node Package Manager (NPM), UUID, Faker.js, MySQL, Sequelize, Bcrypt, Dotenv, Handlebars.js, Bootstrap, and Font Awesome. With secure authentication and efficient database management, CodeThread offers a user-friendly shopping expereince where users can create an account and select their desired size and color to place an order for their cohort t-shirt.
