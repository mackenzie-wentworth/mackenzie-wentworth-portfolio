Peak Fitness

Link to Project: Peak Fitness

As a software developer, I led the development of "Peak Fitness," a dynamic web application that enables users to explore bike trails and access real-time weather conditions for specific locations. Utilizing a combination of HTML, Bulma CSS framework, JavaScript, and JQuery, I crafted an intuitive and visually appealing user interface, ensuring a seamless user experience. My expertise in these technologies facilitated the integration of two server-side APIs: OpenWeather API and TrailAPI. By effectively implementing the OpenWeather API, I provided users with accurate and up-to-date weather information, skillfully handling API requests and dynamically updating weather components on the frontend. Additionally, I seamlessly integrated the TrailAPI to retrieve and display an extensive list of trails for users to explore. Throughout the project, I collaborated closely with my team, participating in design discussions and conducting thorough code reviews to ensure consistent functionality and design. My contributions to "Peak Fitness" showcased my ability to leverage diverse technologies and APIs, delivering an exceptional web application that caters to outdoor enthusiasts seeking bike trails and current weather updates.

JourneyVerse

Link to Project: JourneyVerse

JourneyVerse is a remarkable real-world full-stack MERN application, meticulously crafted from scratch, that introduces a powerful collaboration travel platform. Seamlessly combining Node.js, React.js, Apollo Client, Apollo Server, GraphQL, MongoDB, and JWT Authentication, this app enables users to efficiently manage logistics for their upcoming trips while finding like-minded travel companions. The integration of Apollo GraphQL ensures efficient data communication between the server and client, while JWT Authentication guarantees secure user access. Hosted on Heroku, this feature-rich and user-friendly app showcases our expertise in creating cutting-edge applications that deliver an exceptional travel planning and companionship experience.
